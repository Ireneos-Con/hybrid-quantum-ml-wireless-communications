# Results

Place thesis result artifacts in this directory.

## Suggested Contents

```text
results/
├── figures/
│   ├── bler-vs-snr.png
│   ├── training-loss.png
│   └── model-comparison.png
└── tables/
    ├── experiment-summary.csv
    └── model-results.csv
```

## Figure Checklist

For each figure, include:

- Clear title
- Axis labels
- Units where needed
- Legend
- Short explanation in the main `README.md`

## Table Checklist

For each table, include:

- Model name
- Experiment setting
- Metric values
- Notes about configuration differences
