# Result Data

This directory contains selected final `.npz` result files used for thesis figures and comparisons.

## Layout

```text
results/data/
├── linear-channels/
└── urswipt/
```

## Linear Channels

`linear-channels/` contains final CAE, HAE, BPSK, Hamming, Rayleigh, Rician, and 3GPP comparison data where available.

## UR-SWIPT

`urswipt/` contains selected final UR-SWIPT result arrays.

## Notes

The `.npz` files are lightweight result artifacts, not full training checkpoints. Model checkpoints are intentionally excluded from the repository.
