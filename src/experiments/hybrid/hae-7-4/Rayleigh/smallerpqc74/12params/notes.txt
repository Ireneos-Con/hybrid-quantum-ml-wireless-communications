Smaller PQC design notes for the 7,4 Hybrid Autoencoder

This experiment implements a reduced-parameter version of the original HAE(7,4) quantum encoder while preserving the same overall communication pipeline. The real and imaginary branches are kept separate, the amplitude embedding stage is retained, and the entangling structure remains the same as in the baseline model. The main change is in the trainable single-qubit rotation block.

In the original encoder, each layer used an independent RY rotation parameter for every qubit in both the real and imaginary circuits. For n = 7 qubits and L = 3 layers, this gives 2 x L x n = 42 trainable quantum parameters. In the smaller PQC, we replace the per-qubit parameterization with parity-based parameter sharing: all even-indexed qubits use one shared rotation angle and all odd-indexed qubits use another shared rotation angle, again separately for the real and imaginary branches. As a result, the number of quantum trainable parameters becomes 2 x L x 2 = 12.

The motivation for this modification is to reduce parameter count without changing the encoder so aggressively that its behavior becomes incomparable to the original model. In other words, this is intended to be a controlled simplification rather than a completely new ansatz. The transmitted representation is still generated by amplitude embedding, followed by repeated RY rotations and a nearest-neighbor CNOT chain. Therefore, the comparison with the baseline remains fair: the architecture is structurally similar, but more compact.

The design choice was motivated by the ideas discussed in:

S. Sim, P. D. Johnson, and A. Aspuru-Guzik, "Expressibility and Entangling Capability of Parameterized Quantum Circuits for Hybrid Quantum-Classical Algorithms," Advanced Quantum Technologies, 2019.

That work argues that parameterized quantum circuits should be evaluated not only by application-specific performance, but also by general descriptors such as expressibility and entangling capability. A key implication for this thesis is that increasing circuit flexibility is useful only up to a point. More circuit depth and more trainable parameters do not automatically guarantee a better ansatz. In practice, once a circuit has sufficient expressive power for the task, additional degrees of freedom may offer diminishing returns.

Following that reasoning, the smaller PQC was designed under a "just-enough expressibility" principle. For a hybrid autoencoder, the goal is not to maximize circuit expressibility in the abstract, but to learn a compact signal representation that still supports low BLER after transmission through the channel and decoding. Since the original HAE encoder already uses a low-depth hardware-efficient structure, a natural next step is not to increase complexity, but to test whether part of the parameter freedom can be removed while maintaining similar end-to-end performance.

The odd/even parameter-sharing rule was selected because it provides a simple inductive bias without fully collapsing the circuit into a single shared rotation per layer. It still allows two different local behaviors inside each layer, while substantially shrinking the number of trainable quantum parameters. This makes it a reasonable compromise between expressivity and parameter efficiency.

In summary, this smaller PQC variant was implemented to test whether a more compact ansatz can preserve the practical performance of the original HAE(7,4) encoder while reducing the quantum parameter budget. The experiment is directly inspired by the circuit-design perspective of Sim et al. (2019), especially the idea that useful ansatz design should balance representational power, entangling structure, and implementation cost.
