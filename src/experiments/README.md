# Experiments

This directory contains selected public experiment scripts from the thesis.

## Classical Autoencoder Experiments

`classical/` contains CAE experiments for:

- Rayleigh fading
- Rician fading
- 3GPP fading
- (4, 4), (7, 4), and (8, 8) configurations where available

These scripts support the CAE side of the CAE vs HAE comparisons.

## Hybrid Autoencoder Experiments

`hybrid/` contains HAE experiments for:

- `hae-4-4`
- `hae-7-4`
- `hae-8-8`

The (7, 4) folder also includes low-parameter quantum encoder variants where available.

## UR-SWIPT Experiments

`urswipt/ptfull/` contains the selected UR-SWIPT scripts used for nonlinear-channel information decoding experiments.

## Public Archive Policy

Only source scripts and useful notes are included here. Logs, checkpoints, caches, and machine-specific job scripts are excluded.
